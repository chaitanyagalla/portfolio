## How To Use In Figma

1. Open Figma.
2. Create a blank design file.
3. Drag `figma-import-board.svg` into the canvas.
4. Use the board as a reference while rebuilding editable Figma frames.
5. Recreate the sections using the tokens and component notes in `design-spec.md`.

This is not a native `.fig` file because this workspace does not currently have a Figma connector. The SVG board is still useful as a portable visual reference and can be placed directly in Figma.
